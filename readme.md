## Sobre o projeto

Este projeto é meu portfólio pessoal como desenvolvedor web em início de carreira.

A página reúne meus projetos mais relevantes, tecnologias aplicadas, experiência prática com desenvolvimento full-stack e canais de contato profissional.

O objetivo é servir como uma apresentação direta para oportunidades de estágio e posições júnior em desenvolvimento web.

## Projetos apresentados

### Farofa do Areias

Aplicação full-stack de pedidos para restaurante, com fluxo de cliente, acompanhamento público do pedido e painel administrativo.

**Destaques:**

* Autenticação JWT e rotas administrativas protegidas
* Criação e acompanhamento de pedidos
* Regras de transição de status
* Validações e cálculo de preços no servidor
* PostgreSQL, testes automatizados e CI com GitHub Actions

**Tecnologias:** React, TypeScript, Node.js, Express, PostgreSQL, JWT, Vitest

[Ver repositório](https://github.com/Ryluna19/PFarofa-do-Areias)

---

### Job Application Tracker

Aplicação full-stack para organizar candidaturas de emprego durante o processo seletivo.

**Destaques:**

* Autenticação JWT e rotas protegidas
* CRUD de candidaturas
* Dashboard com estatísticas
* Busca, filtros e atualização de status
* Testes de API e CI com GitHub Actions

**Tecnologias:** React, Node.js, Express, PostgreSQL, JWT, Vitest

[Ver repositório](https://github.com/Ryluna19/PJobApplicationTracker)

---

### Café Curator

Aplicação web que usa a localização do usuário para sugerir cafeterias próximas e organizar locais salvos.

**Destaques:**

* Busca por geolocalização
* Renderização dinâmica de resultados
* Favoritos, notas e status com LocalStorage
* Filtros e ordenação de locais salvos
* Interface responsiva

**Tecnologias:** JavaScript, HTML, CSS, Geolocation API, LocalStorage

[Ver repositório](https://github.com/Ryluna19/PCafe_Proximos)

---

### Music Playlist Manager

Aplicação para criar, organizar e reproduzir playlists, com painel administrativo, busca, filtros e player personalizado.

**Tecnologias:** Node.js, Express, EJS, MongoDB, JavaScript

[Ver repositório](https://github.com/Ryluna19/PMusicPlaylist)

---

### Flask Library System

Sistema de biblioteca com autenticação, persistência de dados e recursos de gerenciamento.

**Tecnologias:** Python, Flask, MySQL, HTML, CSS

[Ver repositório](https://github.com/Ryluna19/PFlask-Library-System)

## Como executar localmente

Clone o repositório:

```bash
git clone https://github.com/Ryluna19/PProfile.git
```

Entre na pasta do projeto:

```bash
cd PProfile
```

Abra o arquivo `index.html` no navegador.
